PYTHON PROGRAMMING

SUBJECT CODE: 120C1A

This book provides a direct path to mastering Python, starting with foundational concepts like data types, control flow, and functions. You will quickly progress to tackling real-world tasks, learning to harness the power of its massive standard library and third-party modules. Whether you're a beginner seeking a powerful first language or an experienced developer looking to expand your toolkit, Python offers the simplicity to learn quickly and the depth to build anything.

OBJECT-ORIENTED PROGRAMMING USING C++

SUBJECT CODE: 120C2A

This book is dedicated to mastering the four pillars of OOP; Encapsulation, Inheritance, Abstraction, and Polymorphism; through the lens of C++. You will learn how to design software by modeling real-world entities using Classes and Objects. Crucially, you will gain hands-on experience with C++; specific concepts like pointers, memory management, constructors/destructors, and virtual functions, ensuring you not only understand what OOP is but how to implement high-performance, rock-solid object-oriented code.

JAVA PROGRAMMING

SUBJECT CODE: 220C4A

This book serves as your comprehensive introduction to the Java landscape. We will start with the fundamental concepts of Object-Oriented Programming (OOP) Classes, Objects, Inheritance, and Polymorphism and quickly progress to advanced topics like Multithreading, Exception Handling, and the Java Collections Framework. You will learn not just the syntax, but the patterns and best practices employed by professional Java developers worldwide, preparing you to tackle everything from desktop applications to complex backend systems.

OPERATING SYSTEM

SUBJECT CODE: 320C5A

This book is your essential guide to understanding this critical layer of software. We begin not with abstract theory, but with the fundamental problem the OS solves: how to manage finite resources (like the CPU, memory, and disk) efficiently, securely, and concurrently for multiple users and countless applications.

RELATIONAL DATABASE MANAGEMENT SYSTEM

SUBJECT CODE: 320C5B

This book introduces you to the core principles and practices of Relational Database Management Systems. We will explore the theoretical concepts of Relational Algebra and Normalization; the rules for designing efficient databases. We then dive into the practical mastery of Structured Query Language (SQL), the universal language for interacting with relational databases. By the end of this journey, you will be able to confidently design effective database schemas, write complex queries for data retrieval, and ensure data integrity using transactions and constraints.

WEB TECHNOLOGY USING PHP

SUBJECT CODE: 320C5C

This book centers on PHP, the powerful language that forms the backbone of millions of websites, including major platforms like WordPress and Facebook. We will guide you through the process of developing a fully functional, data-driven website from the ground up, utilizing the crucial LAMP stack (Linux, Apache, MySQL, PHP). You will learn how to handle user input via forms, manage sessions and cookies, and securely interact with a database, ultimately enabling you to create rich, responsive, and robust web applications.

R-PROGRAMMING

SUBJECT CODE: 320C6A

This book is a comprehensive guide to R, the premier language for statistical computing, data analysis, and graphical visualization. It covers the foundational syntax, core data structures (like vectors, matrices, and data frames), and advanced programming concepts (such as functions, control flow, and object-oriented paradigms). You will learn how to wrangle and clean data, perform various statistical analyses, and create high-quality, professional data visualizations using powerful packages like ggplot2 and the tidyverse. It's essential for anyone entering the fields of data science, statistics, or machine learning.

ADVANCED NETWORKING

SUBJECT CODE: 320C6B

This book takes a pragmatic, in-depth approach to modern network design, configuration, and management, moving beyond basic concepts to focus on enterprise-level challenges. Key topics include advanced routing protocols (like OSPF, EIGRP, and BGP), complex network infrastructure design (VLANs, Layer 3 switching, IP subnetting), and crucial network services (DNS, DHCP). It provides hands-on knowledge of network security, traffic analysis using tools like Wireshark, and technologies like IPv6, preparing administrators and engineers to build, optimize, and secure robust, high-performance wide-area and campus networks.
